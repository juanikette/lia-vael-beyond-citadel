# Character Role Rules

## Normandy Engineering Roles Around Lia

On the Normandy SR-2:

- Gabby Daniels = human engineer
- Kenneth Donnelly = human engineer
- Tali'Zorah = primary quarian anchor

## Typical Mistake to Avoid

Do not use a "new character = interacts equally with everyone" model.

That creates:

- forced dialogue
- redundancy
- fanfic-like tone

## Key Rule

**Not everyone should interact with Lia in the same way.**  
Each character must serve a distinct narrative function.

## Interaction Weight

### Tali — Emotional core (**mandatory**)

- mentor role
- quarian cultural connection
- narrative mirror

Target: ~70% of emotional weight.

### Gabby — Professional clarity

- professional
- clear and direct
- corrects others (especially Ken)
- open, but not emotionally exaggerated

With Lia: Gabby evaluates engineering competence, not vulnerability.

### Ken — Friction, humor, and realism

- jokey
- slightly inappropriate
- unfiltered remarks
- respect grows when he sees real competence

With Lia: jokes/doubt first, respect after demonstrated skill.

## Tone Example (Close to ME2 Voice)

- Ken: "Another quarian engineer, eh? Shepard's got a type."
- Gabby: "Ignore him. If you can keep up with the drive core, you're welcome here."
- Lia: nervous, but firm

This should quickly establish:

- conflict
- support
- team dynamic
- workplace tone

## Framing Rule

Ken and Gabby are not mentors or emotional pillars.  
They are the **workplace environment**.

- Tali = emotional axis
- Gabby/Ken = professional day-to-day texture

This keeps Lia's integration organic and believable.
