# Samples Policy (PCC/TLK)

Este directorio se reserva para muestras reales usadas en pruebas de integracion.

## Objetivo Fase 0

- Dejar preparada la estructura para 1-2 PCC reales conocidos (ME2 OT/LE2).
- Definir nombres esperados para que tests y scripts sean reproducibles.

## Convencion sugerida

- `samples/me2_ot/<archivo>.pcc`
- `samples/le2/<archivo>.pcc`
- `samples/tlk/base/BIOGame_INT.tlk`

## Nota

No subir binarios con restricciones de distribucion sin validacion del equipo.
Si no se versionan, documentar rutas locales equivalentes en pruebas.
